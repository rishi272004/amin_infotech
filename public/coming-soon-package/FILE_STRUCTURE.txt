## Understanding Each File

### 1. **index.html** (755 lines)
**What it is:** Your complete coming soon webpage

**Contains:**
- HTML structure
- Inline CSS styling
- JavaScript for interactions
- All meta tags (SEO)
- Contact section
- Social media links
- Responsive design

**Where it goes:** Root directory (public_html/)
**Upload as:** `index.html`
**File size:** ~30KB
**Editable?** Yes, with text editor

### 2. **.htaccess** (27 lines)
**What it is:** Apache server configuration file

**Does:**
- Redirects all URLs to index.html
- Enables GZIP compression
- Sets up caching (30 days for images, 1 hour for HTML)
- Adds security headers
- Hides file extensions

**Where it goes:** Root directory (public_html/)
**Upload as:** `.htaccess` (with the dot)
**File size:** ~1KB
**Editable?** Only if you know Apache config
**Important:** Enable "Show Hidden Files" to see/upload it

### 3. **README.md** (300+ lines)
**What it is:** Complete reference documentation

**Contains:**
- Feature list
- Technical details
- Customization instructions

**Where it goes:** Keep locally (not on server)
**Purpose:** Reference guide for setup and customization
**Editable?** Yes, for personal notes

## Troubleshooting File Issues

### Can't See .htaccess
- **Problem:** .htaccess is hidden file
- **Solution:** Enable "Show Hidden Files" in cPanel/FTP settings
- In Windows: `View` â†’ `Show hidden files`
- In cPanel: Click `Settings` â†’ Check "Show Hidden Files"

### Logo Doesn't Display
- **Problem:** Wrong file location
- **Solution:** Ensure images folder exists AND logo is inside it
- **Not this:** `public_html/amin_logo.png` âŒ
- **This:** `public_html/images/amin_logo.png` âœ…

### Wrong File Format
- **Problem:** Uploaded as wrong type
- **Solution:** Check file extensions
- `.htaccess` NOT `htaccess.txt`
- `index.html` NOT `index.php` or `index.htm`

### File Seems Corrupted
- **Problem:** Transferred in wrong mode (ASCII vs Binary)
- **Solution:** Use Binary mode for all files in FTP
- **Or:** Re-upload the file
